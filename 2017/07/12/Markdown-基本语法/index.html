关于Markdown

Markdown是一个 Web 上使用的文本到HTML的转换工具，可以通过简单、易读易写的文本格式生成结构化的HTML文档。

  • 简单标记符完成排版，所写即所见，让你专注于文字而不是排版
  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开
  • 格式转换方便，Markdown文本可以轻松转换为 html、pdf等
  • Markdown 的标记语法有极好的可读性
